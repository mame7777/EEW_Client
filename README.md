# EEW_Client
緊急地震速報の情報を取得し，処理するためのパッケージです．  
このパッケージは，地震発生時の情報処理の実験と検討を目的とし開発されています．  
速報の情報の取得には，外部のAPIを利用しています．  
利用者自身の責任で利用し，利用APIの規約を遵守してください．  
本パッケージの利用によるいかなる損害についても，開発者は責任を負いません．
